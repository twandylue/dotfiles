# dotfiles

Add to my dofile repository

```console
git --git-dir ~/Desktop/dotfiles/.git/ --work-tree=$HOME add [specific dotfile]
```

## Brewfile

Generate Brewfile

```console
brew bundle dump --describe --force --file="~/Desktop/Brewfile"
```
